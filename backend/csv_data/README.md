# CSV Data Files for Suppliers

Thư mục này chứa các file CSV với dữ liệu sản phẩm cho từng nhà cung cấp.

## Cấu trúc file:
- `supplier_{ID}_{name}.csv` - File CSV cho từng nhà cung cấp

## Cách sử dụng:
1. Kiểm tra file CSV trong thư mục này
2. Vào trang Import CSV: http://localhost:8080/vnmt/backend/import_csv.php
3. Upload file CSV tương ứng
4. Import vào database

## Lưu ý:
- Mỗi file đã được format đúng theo template
- Tất cả các trường đều đã được điền đầy đủ
- Có thể chỉnh sửa file bằng Excel trước khi import
